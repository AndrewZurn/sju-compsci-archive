Lab 10: Using Interfaces (Work in pairs)

You are given the java file for the FunnyFace class that implements Movable. It presumes a fixed size face that has no color instance variable (all FunnyFace objects are black framed shapes). Your lab will consist of three parts:

Design a Java interface, Reshapable, which supports two methods (call them setWidth and setHeight) that allow a user program to change the width or height of the receiver object.

1.     Modify the definition of the FunnyFace class so that it implements the Reshapable interface in addition to the Movable interface.   As you embark on implementing those methods, keep in mind that as you change the width or height of the face, the eye offset, eye radius, mouth width and mouth height need to change accordingly while maintaining approximately the same relative positions on the face. Those relative positions are defined in the constructor of class FunnyFace.

2.  Implement the changes you have defined in the FunnyFace.java file.

3.     Create a TestFace class which extends FrameWindowController that
