Lab Procedure

Part I

The file RandomColorStarter.java contains the code to generate green lines on the canvas when the user drags the mouse around the canvas (based on the Scribble.java example from class).  To begin, start DrJava and open the file RandomColorStarter.java. This provides the shell of the program you are to write. Rename the class RandomColor and resave the file as RandomColor.java. Compile and run the program by declaring an object of type RandomColor in the interactions pane to see the program work.  Modify the program so that instead of the lines always being green, each time the mouse is pressed, the color is selected randomly from one of red, green, blue, or black, or yellow. You can do this by using the random number generator as in Lab02 but make it choose a number from 1 to 5 and use a selection statement (i.e. if statement) to then set the color based on which number was generated (1 means set the color red, 2 means green, etc.).  Test your program to assure that it works correctly. Show your working program and code to the lab instructor or TA before continuing.

Part II 

Open the file LaundrySorterStarter.java and resave it as LaundrySorter.java. Again, be sure to rename the class LaundrySorter.

As college students, it is important to learn the proper way to sort laundry.  In this part of the lab, you are to write a Java program that will simulate the sorting of laundry. The initial display should display a rectangle (the hamper) that is either white, black, or red to represent white, dark, and colored clothing items. Below this rectangle should be three SLIGHTLY LARGER rectangles to represent three tubs of laundry, Whites, Darks, and Coloreds and given appropriate text labels. The user is to look at the color of the "clothing item" (colored rectangle) in the hamper and click on the appropriate tub. Each time the user correctly selects a tub, the item of clothing will be replaced by another item of random color (black, white, or red). If a wrong selection is made, a message should display regarding the error and the clothing item should be unchanged. You will need to use the method getColor() and an if statement to determine if the colors matched. Also consider using method contains(Location point) available for graphical objects to check if the user clicked inside the correct tub.

In the area above the begin method, you can place all constant and variable declarations. The begin method should place the objects on the canvas to start the clothing process including a random color in the clothing hamper and three rectangles representing the three tubs of clothing. To select the color, generate a random number from 1-3 and based on the value, select a color.

Use constants appropriately for the location and size of the items. You will need to define the conditions that determine which tub was selected by the user (is the mouse location inside the rectangle?) and if the right tub was selected (does the color match?). Note there is the method getColor() for  FilledRects.

Programming Assigment

When you have the lab working correctly, create a new folder inside your Lab03 folder and give it a name similar in format to MillerJohnLab03HW only using your name. Copy all Java files from your Lab3 folder and put them in the newly created folder (but keep the old copies in their place as back up). 

Part 1 - Start by modifying your program to include two integer counters which keep track of correct and wrong selections made for the clothing. Display those counters on the canvas after every user selection.

Part 2 - Then, modify your program to have the user drag the article of clothing from the hamper to the appropriate tub. Class WhatADrag in your book (section 4.3.1 pages 99 and 100) provides a very good example for this purpose. Use onMouseDrag to the move the article of clothing (i.e. hamper) as the user drags and onMouseRelease to tell if the article of clothing should stay in the tub or go back to the hamper. Consider using method overlaps(item) available for graphical objects to check if the user dragged the item into the correct tub.

Part 3 - What if we allowed any color of clothing to be generated randomly? How could we generate a wide range of random colors and how would we determine which tub would be the appropriate tub? As a bonus, allow any random color.
