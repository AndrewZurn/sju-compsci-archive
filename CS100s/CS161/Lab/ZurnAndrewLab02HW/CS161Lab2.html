Lab 2 Using Objects and Numbers - the Turtle and the Hare

Purpose

  • Have students write programs using the objectdraw objects and integer values.
  • Familiarize students with random number generation and use.

Lab Procedure

In this lab, you are to write a Java program that will display a race between two different objects (a rectangle and an oval) that represent the turtle and the hare. With each mouse click, each racer should move a random distance to the right until one of the racers crosses the finish line. To begin, start DrJava and open the file RaceStarter.java.  This provides the shell of the program you are to write. Resave the file as Race.java. 

In the area above the begin method, you can place all constant and variable declarations. The begin method should place the objects on the canvas for the beginning of the race, including a vertical start line near the left edge, a vertical finish line near the right edge, a gray square to represent the hare and a green circle to represent the turtle (these should be placed just to the left of the start line). I expect you to use variable names and constants as appropriate, declaring them in the area above the begin method. In the onMouseClick method, you use randomly generated numbers to move the racers a random amount to the right with each mouse click. Note that one of the methods available to you is the move method. Thus, if I had declared as follows:

    private FilledOval myThing;
  private RandomIntGenerator speed;

and, then, in the begin method created it as follows:

      speed = new RandomIntGenerator(5,10);
      myThing = new FilledOval(10,10,30,20,canvas);

and, then somewhere else I put in:

      myThing.move(speed.nextValue(),0);

then speed.nextValue() would be a random number between 5 and 10 and the FilledOval myThing would move to the right that random amount. Use this pattern for moving both your rabbit and your turtle when the mouse is clicked. You only need one RandomIntGenerator object for both your rabbit and your turtle.

When you are successful in getting the basic racers to work correctly, show your lab instructor or TA.
