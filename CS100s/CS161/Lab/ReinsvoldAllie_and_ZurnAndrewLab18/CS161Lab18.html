Lab 18: Inheritance and extending classes (Work in pairs)

One of the classes provided by objectdraw is the VisibleImage class (see Ch 9 and the ImageEdit code from Lab15).  The constructors and other methods defined for VisibleImage can be found at: http://eventfuljava.cs.williams.edu//library/objectdrawJavadocV1.1.1/ (click on VisibleImage in the left menu).  Your goal is to create a new class called FramedVisibleImage that extends VisibleImage that will not just display the image but will also surround the image with a rectangular n-pixel wide frame (where n is specified in the call to a new FramedVisibleImage via parameter frameWidth). Your new class should include the following constructor:
 
FramedVisibleImage(java.awt.Image image, double x, double y, int frameWidth, DrawingCanvas canvas);


Your FramedVisibleImage should behave appropriately for the following methods which are used in the provided tester class called FramedVisibleImageTester.java:

java.awt.Image getImage()
          Access the Image currently displayed by this VisibleImage 
double getX()
          Get the x coordinate of left edge of frame bounding the image
double getY()
          Get the y coordinate of top edge of frame bounding the image
void move(double dx, double dy)
          Moves the object within the canvas in the x and y direction by the amounts specified as parameters.
void setColor(java.awt.Color c)
          Sets the color of the frame.
void hide()
          Temporarily hides the image with the frame from view on a DrawingCanvas
void show()
          Shows the object if it has been hidden.
void removeFromCanvas()
          Permanently removes the object from the canvas it is currently on.

When done, test your class using the provided FramedVisibleImageTester.java file. You might need to change the size of the canvas in order to see all the output being displayed by the program.
