Lab 12: GUI Without Objectdraw

This lab is to be done individually. Feel free to talk to others in the lab but do your own work.  In this lab you will finish constructing a simple calculator. For the first time, we are doing completely without objectdraw classes. Instead of extending FrameWindowController, we are extending JFrame - the standard GUI building class for the Java Swing GUI-building classes. You will modify the included Calculator.java file to create a simple sort of general purpose calculator which operates on two fields (First and Second) containing numeric values and putting the answer in the Answer field (all of which are JTextFields)

Start by testing it as it is. Load it into DrJava, compile it, and run it. You will see how it works (generally) by putting numbers in the two white fields, clicking on the + button, and seeing the result. Follow the procedure suggested in the comments in file Calculator.java to change Calculator.java to add all the other functionality. If you are wise, you will create the button panel, add the + button to it, and add that panel using contentPane to make sure that works fine. To add another button, add it to the button panel, and try again to see if it works. Then go ahead and add the rest of the buttons to the button panel and check your work.

For functions requiring two numbers like +, -, *, and / both First and Second are used. For example, “-“ means subtract the value from Second from the value from First and put the answer in the answer field. For functions requiring one number like log, exp, sin, cos, and tan the operator works on the value in the first field and displays result in the answer field. +/- First changes the sign of the number in the first field, +/- Second changes the sign of the number in the second field, Clear First and Clear Second set the values in the appropriate field to 0, and Clear All clears both fields. Functions like log, exp, sin, cos, and tan are all in the package Math (hence Math.sin(x)....). You might want to click here to see what the methods in Math are.

Also add a Quit button which stops your program by executing the statement: System.exit(0);

Note: Method focusLost actually sets the values of the numbers first and second to the values in the fields. You need not change this method.
