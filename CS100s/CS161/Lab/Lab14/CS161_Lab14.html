Lab 14: Expanding the Race

As last time, you should again program in pairs.  You should partner with the same person that you partnered with last time.  You should switch roles; whoever was not at the keyboard last time, should be at the keyboard this time.

In this lab, you will complete several classes that will extend the ActiveObject class to get a better handle on how to create classes that have objects that run independently. 

Part I

Start DrJava and open the java files in Lab14 (PopUpInput.java, Rabbit.java, Race.java,  Turtle.java). compile and run the Race. Be sure that you understand how the program works.

You are to modify the files so that a random number of racers (2-8) of random turtles and rabbits, each with their own name can run the race.  Each racer should be set to a random color and a name selected by the user. So one race may have 5 contestants where 4 happen to be turtles and the next race could have 8 racers where 6 happen to be rabbits ( or all rabbits).  The general steps to do this are outlined below.


Create a Racer interface and save it. The methods in the interface should include all the methods in the rabbit and turtle classes  that will be called in the Race class(including the start method!)

Have the rabbit and turtle class implement the Racer interface. Modify the rabbit and turtle classes so that their constructors have a string parameter that holds the name of the racer and sets an instance variable to hold that name (Each contestant will remember it's own name and tell the race its name if it wins).

Declare a constant MAX_RACERS and declare an array of Racers of that size. Declare a variable to hold the number of racers in a given race.
Declare random generators to generate the racer types (a rabbit if 0, a turtle if 1) and to generate random integers to be used to create new colors.

In the begin method, use the PopUpInput to get the number of racers and save the value.

Using a loop, for each racer to be placed in your array, get the name from the user (using PopUpInput), and generate either a rabbit or a turtle in the array slot. Remember to include the name in the call to the constructor.  Generate a random color and set the racer to be that color.

OnMouseClick, have ALL of the racers start.

Compile and run!
