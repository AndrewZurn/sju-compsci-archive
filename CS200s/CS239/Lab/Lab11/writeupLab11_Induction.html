Lab 11: Recursion and Induction


1. Remember good old Fibonacci numbers and their recursive definition?

f0 = 1          f1 = 1        fn = fn-1 + fn-2   (if n>1)

    The claim is that this can be computed by the following formula:

fn = {[(1+sqrt(5))/2]n+1- [(1-sqrt(5))/2]n+1}/sqrt(5)

    You should create methods to compute each of these versions and verify that they seem to compute the same thing.
    (You can do it in Haskell, Java, Perl, or any other language you like.)
    Then prove, inductively, that the second formula is, in fact, a correct formula.
     (The base case in this case is for both 0 and 1.)

Note this is, in fact, different than what I did in class. There we had that f0=0 and f1=1.

2. Now try to show that:
1+x+x2+x3+...+xn = (xn+1- 1)/(x-1)

Do it first by simply writing methods to compute each side of the equation and comparing the results and, then, proving the result inductively.
